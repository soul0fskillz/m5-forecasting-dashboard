## 🚀 What is m5-forecasting-dashboard?

This software helps you forecast demand for thousands of products using machine learning. It processes a large amount of sales data to give you clear predictions. The system uses LightGBM, a fast and accurate forecasting tool. It also has an inventory assistant powered by AI to help you manage stock.

It runs on your Windows computer and provides all features you need to view forecasts and manage inventory through an easy-to-use interface.

## 🚩 About the Technology

The m5-forecasting-dashboard combines several tools:

- **LightGBM:** A machine learning algorithm for fast and accurate forecasts.  
- **FastAPI:** Powers the backend server for quick response.  
- **Next.js:** Manages the user interface in the browser.  
- **PostgreSQL:** Stores demand data and forecasts securely.  
- **Qwen3-32B AI:** Provides inventory advice based on predicted demand.

These parts work together behind the scenes. You only interact with the dashboard in your browser.
